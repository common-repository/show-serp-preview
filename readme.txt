== Description ==

Displays how your page appears in search engine results. A handy tool to design titles and descriptions that are appealing to search engine users.

Your page edit screen will now include a new box "SERP Preview" that displays your content as it will show up when using a search engine such as Google.

The plugin supports a wide variety of themes and plugins that allow you to modify your titles and descriptions, including but not limited to: Yoast SEO, All in One SEO Pack, Jetpack, Genesis, Frugal, and WooFramework.

This plugin is part of [Poor Man's WordPress SEO](http://mokimoki.net/poor-mans-wordpress-seo/) that has also many other features to optimize SEO on your site.
